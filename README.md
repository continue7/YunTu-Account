# 云兔记账

一款界面清新、操作简单的小程序，支持查看任意时间流水、流水统计图、流水导出、预算设置等功能


# 使用技术

* 前端：原生小程序、Lin-UI开源组件库
* 后端：Java、SpringBoot、JPA

目前只开放了前端源码，后端接口我已经部署到了线上，这是[接口文档](https://docs.apipost.cn/view/e4455ed76245f853)，可以直接访问，后续也会开源
